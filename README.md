# 📊 Olist E-commerce Product Analytics Project

## 📌 Overview
This project is a SQL-based product analytics case study using a real-world Brazilian e-commerce dataset (Olist).  
The goal is to analyze customer behavior, sales performance, product trends, and delivery efficiency.

---

## 🛠 Tools Used
- SQL Server / PostgreSQL
- Excel (optional)

---

## 📁 Project Structure

- sql/ → SQL analysis queries
- insights/ → business insights report
- dataset/ → raw data files

---

## 📊 Key Analysis

### Revenue Analysis
- Monthly revenue trends
- Top revenue-generating categories
- Seller performance

### Customer Analysis
- Top cities and states
- Repeat customers analysis

### Product Analysis
- Best-selling products
- Category performance

### Delivery Analysis
- Late delivery orders
- Order status distribution

---

## 💡 Key Insights

- Revenue is concentrated in a few categories
- Repeat customers contribute significantly to sales
- São Paulo leads in customer and order volume
- Some orders experience delivery delays

---

## 🚀 Skills Demonstrated
- SQL (JOIN, GROUP BY, Aggregations, HAVING)
- Data Analysis Thinking
- Business Insights Extraction
